---
title: Get credits
description: Retrieve the remaining credits for a user.
---

## Endpoint
```
GET /v1/jobs/credits/{user_id}
```

## Path parameters

| Parameter | Type | Description |
|---|---|---|
| `user_id` | string | The user ID |

## Response
```json
{
  "user_id": "user_abc123",
  "credits_remaining": 198
}
```

## Example
```bash
curl https://api.motionforge.io/v1/jobs/credits/user_abc123 \
  -H "Authorization: Bearer YOUR_API_KEY"
```
