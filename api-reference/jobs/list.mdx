---
title: List jobs
description: Retrieve all jobs for the authenticated user.
---

## Endpoint
```
GET /v1/jobs
```

## Response
```json
[
  {
    "job_id": "3fa85f64-5717-4562-b3fc-2c963f66afa6",
    "mode": "mode3",
    "status": "done",
    "current_step": "phase_f_delivery",
    "created_at": "2026-03-15T10:00:00Z",
    "updated_at": "2026-03-15T10:02:00Z",
    "output_url": "https://r2.motionforge.io/3fa85f64.mp4",
    "credits_used": 1
  }
]
```

## Example
```bash
curl https://api.motionforge.io/v1/jobs \
  -H "Authorization: Bearer YOUR_API_KEY"
```
