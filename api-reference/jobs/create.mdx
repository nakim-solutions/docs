---
title: Create a job
description: Create a new video generation job.
---

## Endpoint
```
POST /v1/jobs
```

## Request body
```json
{
  "user_id": "string",
  "mode": "mode3",
  "video_url": null,
  "script_text": "Your script here",
  "scenes": [
    {
      "prompt": "A futuristic city at night with neon lights",
      "script_segment": "Welcome to the future of content creation",
      "image_url": null
    }
  ],
  "voiceover_enabled": true,
  "voice_id": "kokoro-en-female-01",
  "music_enabled": true,
  "subtitles_enabled": false
}
```

## Parameters

| Field | Type | Required | Description |
|---|---|---|---|
| `mode` | string | ✅ | `mode1`, `mode2`, or `mode3` |
| `video_url` | string | Mode 1/2 | YouTube URL to analyze |
| `script_text` | string | Mode 3 | Full script narration |
| `scenes` | array | Mode 2/3 | List of scenes |
| `voiceover_enabled` | boolean | ✅ | Enable voiceover |
| `voice_id` | string | If voiceover | Voice to use |
| `music_enabled` | boolean | ✅ | Enable background music |
| `subtitles_enabled` | boolean | ✅ | Enable subtitles |

## Scene object

| Field | Type | Required | Description |
|---|---|---|---|
| `prompt` | string | ✅ | Visual description of the scene |
| `script_segment` | string | Optional | Narration for this scene |
| `image_url` | string | Optional | Reference image URL |


## Example
```bash

  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "mode": "mode3",
    "script_text": "3 reasons to wake up at 5am",
    "scenes": [
      {
        "prompt": "Person waking up at dawn, golden light, cinematic",

      }

    "voice_id": "kokoro-en-female-01",

  }'
```
