## The 3 modes

<CardGroup cols={3}>
  <Card title="Mode 1 — From Video" icon="video" href="/modes/mode1">
    Drop a reference video URL. Motionforge extracts the structure and regenerates a new original video automatically.
  </Card>
  <Card title="Mode 2 — Video + Scenes" icon="sliders" href="/modes/mode2">
    Use a reference video for structure, then customize each scene with your own prompts and images.
  </Card>
  <Card title="Mode 3 — From Script" icon="pen-nib" href="/modes/mode3">
    Start from scratch with your own script and scenes. Full creative control.
  </Card>
</CardGroup>

## Which mode should I use?

| | Mode 1 | Mode 2 | Mode 3 |
|---|---|---|---|
| **Effort** | Minimal | Medium | Full control |
| **Speed** | Fastest | Medium | Depends on you |
| **Creative control** | Low | High | Maximum |
| **Best for** | Quick inspiration | Custom content | Original creation |

## Pipeline

Each mode runs through the same AI pipeline — only the preparation steps differ.
